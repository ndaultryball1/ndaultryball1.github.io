Projects

  • BGK

    My main DPhil project is to investigate the use of deep learning to obtain improved approximate solutions from models of gas dynamics such as the BGK equation. These models can be simpler and quicker to solve, but miss out some of the detailed physics of the Boltzmann description by making simplifications. The goal is to correct for missing physics using machine learning. This involves using neural network closure terms in the equation, and deriving appropriate adjoint equations which are necessary to train the networks.

    Collaborators: Justin Sirignano, Jonathan MacArt, Marco Panesi

  • DSMC

    I am also using machine learning to enhance DSMC algorithms for the Boltzmann equation.

  • Deep Learning
